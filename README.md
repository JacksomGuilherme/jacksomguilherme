## Projetos em destaque

### Scheduler — Sistema Web de Agendamento de Aulas e Treinamentos

Sistema web desenvolvido para automatizar o agendamento de aulas de reposição e treinamentos, substituindo controles manuais feitos em planilhas.

**Principais funcionalidades:**

- Cadastro de cursos, categorias, horários e professores.
- Controle de acesso por nível hierárquico.
- Regras de disponibilidade de professores.
- Validação de competências por aula.
- Bloqueio de vagas conforme capacidade da sala.
- Redução de erros operacionais de 80% para 20%.

### Spotify Controller para E-readers

Aplicação web otimizada para e-readers, permitindo controlar o Spotify pelo Kindle ou por dispositivos com navegação limitada.

**Principais funcionalidades:**

- Autenticação via QR Code.
- Integração com a API do Spotify.
- Autenticação com OAuth 2.0.
- Gerenciamento seguro de sessões com cookies.
- Criptografia AES.
- Exibição de playlists, álbuns e artistas seguidos.
- Controles de reprodução: play, pause, avançar e retroceder.
